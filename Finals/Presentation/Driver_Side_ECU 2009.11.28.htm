Requirements

  • D0: Prior to the car being started, the Driver Side ECU is inactive, therefore all signals to the Driver Side ECU other than the engine start signal (S) is ignored.
  • D1: Any car start signal recieved after the initial car start signal will act as a ECU reset.
  • D2: When the mirror is toggled for the Passenger Side Mirror, the mirror adjustment signals, are forwarded to the Passenger Side ECU via the CAN bus.
  • D3: The state of the ECU must be updated if the state of the engine changes.
  • D4: Extraneous temperature and timer signals are ignored.
  • R1: There is a switch that toggles for adjustment of either the driver side or the passenger side mirror.
  • R2: The driver side ECU initilizes when the car key is in start position.
  • R3: Each mirror can be adjusted in in the veritcal and has extreme up and down positions. Each mirror can be adjusted in the horizontal positions and has extreme positions.
  • R4: If requsted movement cannot be made because the mirror is already in an extreme position, an error message is generated and sent via the CAN bus.
  • R5: The mirrors have heaters that come on when the outside temperature is 2 degrees Celsius, stay on for ten seconds, and stay off for two minutes before coming on again.
  • R6: Engine has to be running when mirror heating is used.
  • R7: The engine state information is sent via the MRUN signal (MRUN = 1 when engine is on).

Interfaces

  • CAN Bus
    • Description: Central commication header for all the different Electronic Controlling Units in the vehicle.
    • Requirement Trace: [ R4 ]
  • Driver Side Mirror
    • Description: This components includes the mirror, the mirror motor, the mirror heater coil, extreme up/down/left/right postition sensors, and two timers.
    • Requirement Trace: [ D2 ]
  • Mirror Adjustment Panel
    • Description: This panel houses is located next to the steering wheel and it houses the mirror adjustment stick and the mirror toggle switch.
    • Requirement Trace: [ R1 ]

  • Extreme Down
    • Description: Mirror has reached the extreme down position. All downward inputs will be ignored
    • Requirement Trace: [ R4 ]
  • Extreme Left
    • Description: Mirror has reached the extreme left position. All leftward inputs will be ignored
    • Requirement Trace: [ R4 ]
  • Extreme Right
    • Description: Mirror has reached the extreme right position. All rightward inputs will be ignored
    • Requirement Trace: [ R4 ]
  • Extreme Up
    • Description: Mirror has reached the extreme up position. All upward inputs will be ignored
    • Requirement Trace: [ R4 ]
  • MRUN
    • Description: false if the engine if off true if the engine is running
    • Requirement Trace: [ R7 ]
