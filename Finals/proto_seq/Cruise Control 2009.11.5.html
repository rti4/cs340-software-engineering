Requirements

  • D10resume: A cruise control can only be resumed if it was activated atleast once.
  • D11: Cruise Control is taken out of passive mode when the accelerator is released.
  • D12: The cruise control system will only respond to the activate button when the cruise control system if off.
  • D13: Pressing the CC_DECREMENT button decreases the set cc_speed by 1 kph, and CC_INCREMENT button increments the set cc_speed by 1 kph
  • D1passive: If in cruise mode, pressing the accelerator puts cruise control in passive mode
  • D2~display: If cruise control is not active, red indicator light is off and the cruise speed is not displayed on the dashboard
  • D3~passive: if in passive mode, releasing the accelator takes cruise control out of passive mode
  • D4caroff: All inputs are non responsive till the car has been started.
  • D5ccinactive: When the cruise control system is off or deactivated, all inputs except c_active or c_resume, which can activate the cruise control system, are ignored.
  • D6resume: The cruise control can only be resumed by c_resume when the speed-to-be-maintained is set.
  • D7ccreset: Once the car has been started, additional ignition starts will act as a cruise control system reset.
  • D8release: In cruise control active mode, the brake release and accelerator release events cause no response. In cruise control passive mode, the brake release and accelerator press events cause no response.
  • D9passive: In passive mode, the cruise control system will not try to control the speed of the car since the driver is pressing the accelerator.
  • R1highspeed: When active, the system keeps the vehicle traveling steadily at the driver-specified high speed, without the driver having to keep his foot on the accelerator pedal.
  • R2>50: The cruise control system can be activated only if the car is traveling at least 60 km/hr and if the brake is not being pressed.
  • R3brake: An active cruise control system is automatically deactivated if the driver engages the brake (presumably to react to road or traffic conditions), or the accelerator pedal
  • R4resume: The driver can resume a deactivated cruise control system.
  • R5carOff: The system terminates when the car's ignition is turned off.
  • R6ccActive: The speed-to-be-maintained is initially the current speed of the vehicle when cruise control is activated.
  • R7deactivate: if the system is deactivated. When cruise control Is resumed, the speed-to-be-maintained will be whatever value it was when the cruise control system was last active. The command to resume cruise control will succeed only if the brake pedal is not being pressed.
  • R8display: When cruise control is activated, a red indication light is turned on, and the cruise speed is displayed on the dashboard.

State Variables

  • Car State
    • Description: Is the car on or off
    • Possible Values: [ ----- , Off , On ]
  • Cruise Control Mode
    • Description: Off means it has never been set Active means it has been set Passive means it has been set, but the accelarator is being pressed Deactive means it has been set, but is currently deactivated
    • Possible Values: [ Passive , Deactive , ----- , Off , Active ]
